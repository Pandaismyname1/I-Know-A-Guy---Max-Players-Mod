# Max Players Mod

Increase the maximum lobby player count in **I Know a Guy** beyond the default limit of 4.

## Features

- Configurable max players (default: 8)
- Extended Create Lobby dropdown (shows 2 up to your configured max)
- Spawn point wrapping for extra players
- Optional F5 debug diagnostics (disabled by default)

## Installation

1. Install this mod via Thunderstore Mod Manager, or manually place `IKnowAGuyMaxPlayersMod.dll` in your game's `Mods/` folder
2. Launch the game — the mod loads automatically

## Configuration

Edit `UserData/MelonPreferences.cfg` after first launch:

```ini
[MaxPlayersMod]
MaxPlayers = 8
DebugMode = false
```

## Requirements

- MelonLoader v0.7.x
- I Know a Guy (Demo or Full)
